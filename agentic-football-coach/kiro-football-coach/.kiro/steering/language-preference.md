# Language Preference

When this steering file is active, respond to the participant in their preferred language (Japanese or Korean). Use the translated key terms below to keep workshop terminology consistent.

## Instructions

- Respond entirely in the participant's chosen language.
- Use English for code identifiers, CLI commands, file names, and API names (e.g., `main.py`, `agentcore`, `create_react_agent`).
- Use the translated terms below for workshop concepts and game terminology.
- Reference content from `content-reference/ja/` or `content-reference/ko/` as appropriate.
