# フットボールの戦略と戦術

AIフットボールエージェントのための攻撃戦略、守備戦略、戦術プログラミング概念の包括的リファレンス。

---

## 攻撃戦略

### ポゼッションプレー

- 多くのショートパスでボールをキープ
- ボールを追わせることで相手を疲れさせる
- 完璧な攻撃機会を待つ
- **エージェントの行動：** 忍耐強いパス、フォーメーション維持、リスクの高いフォワードボールより安全なパスを優先

### カウンターアタック

- 深く守り、プレッシャーを吸収
- ボールを奪ったら素早く攻撃
- スピードを使って相手のポジション崩れを突く
- 直接的で速いフォワードパス
- **エージェントの行動：** 守備から攻撃への素早い切り替え、ボール奪取時に前方へスプリント、相手守備ラインの裏のスペースを突くロングパス

### ハイプレス

- 相手がボールを持ったらすぐにプレッシャーをかける
- 相手陣内でボールを奪おうとする
- 高いスタミナと調整が必要
- **エージェントの行動：** ピッチ高い位置での積極的なポジショニング、素早いタックル、パスコースを塞ぐ、ユニットとしてプレストリガーを調整

### ウイングプレー

- フィールドのサイドから攻撃
- ペナルティボックスにボールをクロス
- 相手の守備を水平方向に広げる
- **エージェントの行動：** ウイングのワイドなポジショニング、フルバックのオーバーラップラン、ボックスへのクロス、中央にポジションを取るフォワードをターゲット

## 守備戦略

### ゾーンディフェンス

- 各プレイヤーがフィールドの特定エリアを守る
- 相手の動きに関係なくチームの形とスペーシングを維持
- **エージェントの行動：** 割り当てられたポジションを保持、指定ゾーンをカバー、ボールの動きに合わせてユニットとしてシフト、個々の相手を追うよりゾーンの整合性を優先

### マンツーマンマーキング

- 各ディフェンダーが特定の相手を追跡
- 割り当てられたプレイヤーをどこに移動しても密接にフォロー
- **エージェントの行動：** 特定の相手IDをマーク、動きに追従、タックル距離内に留まる、明示的に再割り当てされた場合のみ切り替え

### コンパクトディフェンス

- すべてのプレイヤーをタイトなブロックで密集させる
- 相手がライン間のスペースを見つけるのを困難にする
- **エージェントの行動：** タイトなフォーメーション維持、プレイヤー間のギャップを削減、守備とミッドフィールド間のスペースを圧縮、相手をサイドに追いやる

### オフサイドトラップ

- ディフェンダーが一緒に前進して攻撃者をオフサイドにする
- すべてのディフェンダー間の完璧なタイミングと調整が必要
- **エージェントの行動：** トリガー（例：相手がパスしようとしている）に合わせた同期した前進、すべての相手ポジションの認識、ラインとしてステップアップ

## 戦術プログラミング概念

これらの概念はエージェントの意思決定ロジックに直接変換されます。

### ボールポゼッション判断

| 状況 | 推奨アクション |
|---|---|
| 安全な位置、良いパスオプションなし | ボールをキープ、ドリブルまたはホールド |
| 味方がより良い位置にいる | パス（ショートまたはスルー） |
| 相手からプレッシャーを受けている | 素早くパスまたはクリア |
| 攻撃を作っている | フォワードパスまたはサイドチェンジ |
| ゴールに近い、クリアなシュート、良い角度 | シュート |
| ゴールに近い、シュートがブロックされている | パスを探すまたはリポジション |

### ボールなしの動き

- **ランニング：** パスを受けるためにオープンスペースに移動；オフサイドにならないようにランのタイミングを合わせる
- **スペース作り：** ディフェンダーをキーエリアから引き離して味方のギャップを開く
- **ボールキャリアのサポート：** パスオプションを提供するために近くにポジションを取る
- **ポジション回復：** ポゼッションを失った時に守備の形にトラックバック

### チーム調整パターン

- **パストライアングル：** 3人のプレイヤーが三角形を形成して簡単なショートパスオプションを作る；プレーの移動に合わせて三角形を回転
- **オーバーラップラン：** フルバックまたはミッドフィルダーがボールキャリアの外側を追い越してフォワードパスを受けクロスを上げる
- **サイドチェンジ：** 相手が薄いフィールドの反対側へのロングパスでスペースを活用
- **サードマンラン：** プレイヤーAがプレイヤーBにパス、プレイヤーBがランニングするプレイヤーCにレイオフ — 最初のプレッシャーラインをバイパス

### ゲーム状態の認識

| ゲーム状態 | 戦術調整 |
|---|---|
| **勝っている** | ポゼッションに集中、リードを守る、時間を管理、リスクを減らす |
| **負けている** | よりリスクを取る、高くプレス、より頻繁にシュート、追加プレイヤーを前方にコミット |
| **引き分け** | バランスの取れたアプローチ、機会を探す、過度にコミットしない |
| **終盤 + 勝っている** | 時間管理、相手陣内でボールをキープ、不必要なファウルを避ける |
| **終盤 + 負けている** | 全力攻撃、セットプレーでゴールキーパーも上がれる、守備リスクを受け入れる |

### スタミナ管理

- **賢くスプリント：** 重要な瞬間のためにスプリントを温存（ブレイクアウェイ、最後のタックル、得点チャンス）
- **ローテーション：** プレイヤーが交代でプレスと回復；同じプレイヤーに毎ティックスプリントさせない
- **ポジショニング：** 良いポジショニングは走る必要を減らす — ボールが来る前に正しい場所にいる
- **可能な時はWALK：** 時間的プレッシャーがない時はRUNの代わりにWALKを使用してスタミナを節約
- **現在のスタミナを監視：** 残りスタミナに基づいてアグレッシブさを調整；疲れたプレイヤーは遅く、効果が低い

## 戦略とエージェントのマッピングサマリー

| 戦略 | 主要アクション | スタミナ影響 | リスクレベル |
|---|---|---|---|
| ポゼッションプレー | SHORT_PASS、DRIBBLE、WALK | 低 | 低 |
| カウンターアタック | SPRINT、LONG_PASS、THROUGH_PASS、SHOOT | 高（バースト） | 中 |
| ハイプレス | RUN、TACKLE、INTERCEPT、SPRINT | 高（持続） | 中〜高 |
| ウイングプレー | RUN、CROSS、SPRINT、DRIBBLE | 中 | 中 |
| ゾーンディフェンス | WALK、STOP、INTERCEPT、MARK | 低 | 低 |
| マンツーマンマーキング | RUN、MARK、TACKLE | 中 | 中 |
| コンパクトディフェンス | WALK、STOP、MARK、INTERCEPT | 低 | 低 |
| オフサイドトラップ | RUN（同期）、STOP | 低〜中 | 高 |
