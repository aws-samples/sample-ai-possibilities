# 풋볼 전략과 전술

AI 풋볼 에이전트를 위한 공격 전략, 수비 전략, 전술 프로그래밍 개념의 종합 참조 자료.

---

## 공격 전략

### 점유율 플레이

- 많은 짧은 패스를 통해 볼을 유지
- 상대를 쫓게 만들어 지치게 함
- 완벽한 공격 기회를 기다림
- **에이전트 행동:** 인내심 있는 패스, 포메이션 유지, 위험한 전방 패스보다 안전한 패스 우선

### 역습

- 깊게 수비하고 압박을 흡수
- 볼을 빼앗으면 빠르게 공격
- 속도를 사용하여 상대의 포지션 붕괴를 이용
- 직접적이고 빠른 전방 패스
- **에이전트 행동:** 수비에서 공격으로의 빠른 전환, 볼 회수 시 전방으로 스프린트, 상대 수비 라인 뒤 공간을 활용하는 롱패스

### 하이 프레스

- 상대가 볼을 가지면 즉시 압박
- 상대 진영에서 볼을 빼앗으려 함
- 높은 스태미나와 조정 필요
- **에이전트 행동:** 피치 높은 곳에서의 공격적 포지셔닝, 빠른 태클, 패스 경로 차단, 팀 단위로 프레스 트리거 조율

### 윙 플레이

- 필드 측면에서 공격
- 페널티 박스로 볼을 크로스
- 상대 수비를 수평으로 늘림
- **에이전트 행동:** 윙어의 넓은 포지셔닝, 풀백의 오버래핑 런, 박스 안으로 크로스, 중앙에 위치한 공격수를 타겟

## 수비 전략

### 지역 수비

- 각 선수가 필드의 특정 영역을 수비
- 상대 움직임에 관계없이 팀 형태와 간격 유지
- **에이전트 행동:** 할당된 포지션 유지, 지정된 구역 커버, 볼 이동 시 팀 단위로 이동, 개별 상대 추적보다 구역 무결성 우선

### 대인 마크

- 각 수비수가 특정 상대를 추적
- 할당된 선수를 어디로 이동하든 밀접하게 따름
- **에이전트 행동:** 특정 상대 ID 마크, 움직임 추종, 태클 거리 내 유지, 명시적으로 재배정될 때만 전환

### 컴팩트 수비

- 모든 선수를 타이트한 블록으로 가깝게 유지
- 상대가 라인 사이에서 공간을 찾기 어렵게 만듦
- **에이전트 행동:** 타이트한 포메이션 유지, 선수 간 간격 축소, 수비와 미드필드 사이 공간 압축, 상대를 측면으로 유도

### 오프사이드 트랩

- 수비수가 함께 전진하여 공격수를 오프사이드로 만듦
- 모든 수비수 간 완벽한 타이밍과 조정 필요
- **에이전트 행동:** 트리거 시 동기화된 전진 (예: 상대가 패스하려 할 때), 모든 상대 위치 인식, 라인으로 전진

## 전술 프로그래밍 개념

이러한 개념은 에이전트 의사 결정 로직으로 직접 변환됩니다.

### 볼 점유 결정

| 상황 | 권장 액션 |
|---|---|
| 안전한 위치, 좋은 패스 옵션 없음 | 볼 유지, 드리블 또는 홀드 |
| 팀원이 더 나은 위치에 있음 | 패스 (숏 또는 스루) |
| 상대의 압박을 받고 있음 | 빠르게 패스하거나 클리어 |
| 공격 전개 중 | 전방 패스 또는 사이드 전환 |
| 골대에 가까움, 명확한 슛, 좋은 각도 | 슛 |
| 골대에 가까움, 슛이 막힘 | 패스를 찾거나 재포지셔닝 |

### 볼 없이 움직이기

- **런 만들기:** 패스를 받기 위해 열린 공간으로 이동; 오프사이드에 걸리지 않도록 런 타이밍 조절
- **공간 만들기:** 수비수를 핵심 영역에서 끌어내어 팀원을 위한 간격 열기
- **볼 캐리어 지원:** 패스 옵션을 제공하기 위해 근처에 포지셔닝
- **포지션 복귀:** 점유를 잃으면 수비 형태로 복귀

### 팀 조정 패턴

- **패스 삼각형:** 세 선수가 삼각형을 형성하여 쉬운 숏패스 옵션 제공; 플레이 이동에 따라 삼각형 회전
- **오버래핑 런:** 풀백이나 미드필더가 볼 캐리어를 지나 외측으로 달려 전방 패스를 받고 크로스 전달
- **사이드 전환:** 상대가 얇은 반대편으로 롱패스하여 공간 활용
- **서드맨 런:** 선수 A가 선수 B에게 패스, 선수 B가 런을 하는 선수 C에게 레이오프 — 첫 번째 압박 라인 우회

### 게임 상태 인식

| 게임 상태 | 전술 조정 |
|---|---|
| **이기고 있을 때** | 점유율에 집중, 리드 보호, 시간 관리, 위험 감소 |
| **지고 있을 때** | 더 많은 위험 감수, 높게 압박, 더 자주 슛, 추가 선수를 전방에 투입 |
| **비기고 있을 때** | 균형 잡힌 접근, 기회 찾기, 과도한 투입 금지 |
| **경기 후반 + 이기고 있을 때** | 시간 관리, 상대 진영에서 볼 유지, 불필요한 파울 피하기 |
| **경기 후반 + 지고 있을 때** | 전면 공격, 세트피스 시 골키퍼도 올라감, 수비 위험 감수 |

### 스태미나 관리

- **현명하게 스프린트:** 중요한 순간을 위해 스프린트 절약 (돌파, 최후의 태클, 득점 기회)
- **로테이션:** 선수들이 교대로 압박과 회복; 같은 선수가 매 틱마다 스프린트하지 않도록
- **포지셔닝:** 좋은 포지셔닝은 달리기 필요성 감소 — 볼이 오기 전에 적절한 위치에 있기
- **가능하면 걷기:** 시간 압박이 없을 때 RUN 대신 WALK를 사용하여 스태미나 절약
- **현재 스태미나 모니터링:** 남은 스태미나에 따라 공격성 조절; 지친 선수는 더 느리고 덜 효과적

## 전략-에이전트 매핑 요약

| 전략 | 주요 액션 | 스태미나 영향 | 위험 수준 |
|---|---|---|---|
| 점유율 플레이 | SHORT_PASS, DRIBBLE, WALK | 낮음 | 낮음 |
| 역습 | SPRINT, LONG_PASS, THROUGH_PASS, SHOOT | 높음 (버스트) | 중간 |
| 하이 프레스 | RUN, TACKLE, INTERCEPT, SPRINT | 높음 (지속) | 중간-높음 |
| 윙 플레이 | RUN, CROSS, SPRINT, DRIBBLE | 중간 | 중간 |
| 지역 수비 | WALK, STOP, INTERCEPT, MARK | 낮음 | 낮음 |
| 대인 마크 | RUN, MARK, TACKLE | 중간 | 중간 |
| 컴팩트 수비 | WALK, STOP, MARK, INTERCEPT | 낮음 | 낮음 |
| 오프사이드 트랩 | RUN (동기화), STOP | 낮음-중간 | 높음 |
